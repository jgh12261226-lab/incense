# [PRD] 힙한 감성의 히든 메시지 인센스 브랜드 랜딩 페이지 구축

## 1. 프로젝트 개요

- **프로젝트명:** 힙한 감성의 히든 메시지 인센스 홍보 랜딩 페이지
- **목표:** 템플릿 중심의 흔한 커머스 공식을 탈피하고, 고블린 스튜디오풍의 통통 튀는 볼드 폰트와 라인프렌즈 인터랙티브 북 스타일의 입체적인 팝업 플립 모션을 결합하여 ‘이벤트에 진심인’ 2030 타깃에게 강력한 도파민과 구매 확신을 선사함.
- **핵심 가치:** 타들어 가는 붉은 불씨 끝에 마주하는 시각적 반전, 공간에 각인되는 스모키한 고농축 잔향, 인테리어 오브제(인센스 홀더)로서의 가치를 하나의 장난감을 조작하는 듯한 게미피케이션(Gamification) 경험으로 전달.

## 2. 디자인 및 무드 시스템 (2026 트렌드 반영)

### 2.1 색상 팔레트 (Neo-Retro Luxury & Smoke)

- **배경색 (Background):** `#161512` (에보니 스모크 / Ebony Smoke) - 갈색조와 잿빛이 도는 깊고 따뜻한 밤의 색.
- **주조색 (Primary):** `#EAE4D9` (언블리치드 페이퍼 / Unbleached Paper) - 재생지 가공 텍스처를 닮은 부드러운 화이트.
- **강조색 (Accent):** `#FF6B35` (네오 테라코타 / Neo Terracotta) - 타오르는 인센스 불씨의 온도를 담은 채도 높은 오렌지.

### 2.2 타이포그래피 System

- **국문 헤드라인:** 모서리가 둥글고 두툼한 볼드 폰트 (예: 산돌 뭉치 등 상업용 폰트 임베딩)
- **영문 헤드라인:** 유기적이고 덩어리감이 살아있는 3D 젤리 형태의 서체 (예: Instrument Serif 또는 Marck 스타일)
- **본문용:** `Geist Sans` 또는 `Pretendard 가변 폰트` (가독성과 자간 비율 극대화)

## 3. 섹션별 상세 기획 및 레이아웃 정의

### SECTION 1: 히어로 섹션 (Hero - The Trigger)

- **목적:** 사이트 진입 후 3초 이내에 뻔하지 않은 몰입감을 선사하고, 사용자의 즉각적인 조작을 유도함.
- **레이아웃:** 화면 중앙에 고정(Sticky)된 3D 스타일의 거대한 '인센스 북' 오브제가 닫혀 있고, 상단에 볼드한 헤드라인 배치. 하단에 마우스 스크롤 유도 인디케이터 배치.
- **들어갈 요소:** 메인 타이포그래피, 닫힌 3D 인센스 북 일러스트/소스, 스크롤 유도 애니메이션 아이콘.
- **헤드라인 카피:** `"불을 붙여 태워봐야 진짜를 알지. 째깍째깍, 진심이 타들어 가는 중 ⏱️"`

### SECTION 2: 향의 지속성 (The First Scent)

- **목적:** 공간에 은은하게 스며 오래 머무는 스모키 고농축 잔향이라는 물리적 제품 스펙을 감성적으로 전달.
- **레이아웃:** 2분할 레이아웃. 좌측에는 인터랙티브 북의 첫 번째 장이 열리는 모션 영역, 우측에는 향의 노트를 감각적으로 시각화한 잡지 스타일의 타이포그래피 그리드 배치.
- **들어갈 요소:** 향 노트 데이터, 아지랑이 연기 입자를 시각화한 미세 그라디언트 텍스처 칩.
- **헤드라인 카피:** `"연기가 사라진 다음 날 아침까지, 그날의 잔향은 방 안에 머뭅니다."`

### SECTION 3: 시각적 반전 (The Hidden Story)

- **목적:** 인센스가 타들어가며 연기 너머로 메시지가 떠오르는 극적인 경험을 간접 체험하게 함.
- **레이아웃:** 화면 전체가 인센스 홀더와 스틱의 상단 단면 뷰(Top-view)로 고정되며, 사용자의 스크롤 진척도에 따라 가로 방향으로 짙은 연기 레이어가 바람에 날려 걷히는 서사적 구성.
- **들어갈 요소:** 단계별 인센스 연기 흩날림 이미지/CSS 마스킹 소스, 타오르는 불씨 그래픽, 숨겨진 메시지 폰트 영역.
- **헤드라인 카피:** `"쉿! 향이 스르륵 타들어가면, 연기 너머 숨겨둔 속마음이 쨘- 하고 나타나! ✨"`

### SECTION 4: 오브제 가치 & 종장 (The Perfect Epilogue)

- **목적:** 인테리어 소품으로서의 가치를 증명하고 최종 구매(커스텀) 전환 유도.
- **레이아웃:** 하이엔드 무드의 세라믹 인센스 챔버 및 제품 패키지 배치가 그리드로 깔리며, 스크롤 종점에 도달 시 화면 중앙에 거대하고 탱글한 젤리 스타일의 CTA 버튼이 등장함.
- **들어갈 요소:** 패키지 및 챔버 목업 이미지, 최종 가치 제안 카피, 볼드 인터랙션 CTA 버튼.
- **헤드라인 카피:** `"이제 당신이 숨겨둔 진심을 피울 차례입니다."`

## 4. 모션 플로우 및 인터랙션 정의

### 4.1 섹션별 3단계 모션 매트릭스

| 섹션          | 진입 시 (In-view)                                                                        | 스크롤 시 (On-scroll)                                                                            | 호버 시 (Hover / Mouse-move)                                                                                     |
| :------------ | :--------------------------------------------------------------------------------------- | :----------------------------------------------------------------------------------------------- | :--------------------------------------------------------------------------------------------------------------- |
| **SECTION 1** | 헤드라인 글자가 팝업북처럼 앞으로 **"뿅!"** 튀어나오며 위아래로 미세하게 흔들림(Jiggle). | 스크롤 속도에 맞춰 인센스 북의 표지가 입체적으로 탄성 있게 살짝 들썩임.                            | 마우스 커서가 붉은 인센스 불씨 이펙트로 변형되어 주변에 주황색 연기 잔상을 남김.                                             |
| **SECTION 2** | 우측 향기 그리드 텍스트가 좌측에서 우측으로 밀리며 순차적으로 등장.                      | 첫 번째 책장이 고무줄처럼 튕기며 **"팅!-"** 하고 입체적으로 찰지게 넘어감.                       | 향기 칩 위에 마우스를 올리면 칩이 연기 풍선처럼 부풀어 오름.                                                          |
| **SECTION 3** | 배경의 암전도가 깊어지며 메인 문구가 센터에 정렬됨.                                      | 스크롤을 내릴수록 불투명한 인센스 연기 레이어가 흩어지며 바닥의 글자가 **Blur 100%에서 0%로 또렷해짐**. | 마우스를 움직이는 자리를 따라 연기 레이어가 실시간으로 일렁이는 왜곡(Distortion) 발생.                           |
| **SECTION 4** | 최종 젤리 CTA 버튼이 화면 한가운데에 우뚝 솟아오름.                                      | 뒷표지가 완전히 닫히며 화면의 스크롤이 고정 해제됨.                                              | 버튼이 아래로 살짝 **찌러졌다가 위로 탱글하게 튕겨 오름(Squish & Bounce)**. 화살표(`→`)가 우측으로 두 번 탭핑함. |

### 4.2 인터랙션 포인트 (Interaction Points) 목록

- **마우스 커서 커스텀:** 사이트 전체에서 마우스 커서는 동그란 주황색 불씨와 연기 흔적으로 대체되며, 인센스의 '스모키한 열기'를 시각화함.
- **스크롤 스티키 플립:** 인터랙티브 북 구현을 위해 특정 섹션 진입 시 스크롤 플로우가 고정(`position: sticky`)되고, 스크롤 진척도(`progress`)가 0에서 1이 될 때까지 책장 넘김과 연기 리빌 모션이 강제 연동됨.
- **클릭(CTA):** 최종 CTA 버튼 클릭 시 짙은 주황빛 연기가 화면 전체를 빠르게 메우는 스모크 팝(Smoke Pop) 효과와 함께 다음 페이지 레이아웃으로 유기적으로 전환됨.

## 5. 모바일 대응 가이드 (Responsive Design)

- **레이아웃 스위칭:** PC에서의 좌우 2분할 구조와 3D 책장 플립 모션은 모바일 환경에서 상하 스택(Stack) 구조로 변경하되, 책장이 위에서 아래로 껍질이 벗겨지는 **'수직 스티키 스태킹(Vertical Sticky Stacking)'** 모션으로 변환함.
- **인터랙션 대체:** 터치 디바이스 특성상 마우스 호버(Hover) 이펙트가 작동하지 않으므로, **화면 터치(Touch Start) 시** 터치한 지점을 중심으로 연기 아지랑이가 피어오르는 일렁임 이펙트가 0.5초간 발생 후 사라지도록 대체 처리.
- **타이포그래피 및 요소 축소:** 굵고 통통 튀는 볼드 폰트는 모바일 화면 크기에 맞춰 자간을 좁히고 폰트 사이즈를 가변(`vw` 단위) 처리하여 글자 깨짐 및 불필요한 줄바꿈 방지. 버튼의 탱글한 탄성 모션은 터치 후 손을 떼는 순간(`Touch End`) 구동되도록 매핑.

## 6. 개발 명세 (Technical Specifications)

### 6.1 기술 스택

- **Framework:** React 18+
- **Build Tool:** Vite
- **Styling:** 일반 CSS (Vanilla CSS / CSS Modules)
- **Motion/Animation:** 수순 CSS `@keyframes`, `Transition`, `Transform` 및 React `Intersection Observer API` 기반 스크롤 트리거링 (라이브러리 배제, 순수 CSS와 React 상태 관리를 통한 최적화)
