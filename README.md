# 🕯️ 캔틀 (Cantle) - 프리미엄 향초 브랜드 스토리 & 커머스 웹 사이트

> **"연기가 사라진 다음날 아침까지, 그날의 잔향은 온전히 공간에 스며들어 머무릅니다."**
> 
> 캔틀(Cantle)은 맑고 알싸한 나무 향부터 은은하게 가라앉는 패출리와 미르의 잔향까지, 프리미엄 캔들이 선사하는 감성적인 향의 여정을 극적인 3D 인터랙션과 우아한 타이포그래피로 연출한 스토리텔링형 커머스 웹 애플리케이션입니다.

---

## ✨ 핵심 차별화 기능 (Key Features)

### 📖 1. 3D 인터랙티브 북플립 (Interactive 3D Book Flip)
향의 노트를 한 권의 빈티지 가죽 서적으로 엮어낸 인터랙티브 3D 뷰포트입니다.
* **180도 평평한 완전 밀착형 정렬**: 책장이 넘어갈 때 공중에 부자연스럽게 뜨지 않고, 바닥면과 앞선 장 위에 정갈하게 착 달라붙어 포개지는 평평한 수평 180도 플립(`rotateY(-180deg)`)을 구현했습니다.
* **초슬림 Z-depth 적층 기술**: 화면 깨짐(Z-fighting) 현상을 완벽히 방지하면서도 얇은 종이의 한 장 두께감만을 살려주는 `1.5px` 정밀 Z축 적층 엔진을 장착했습니다.
* **반동이 배제된 프리미엄 Ease-Out 모션**: 넘길 때 뒤로 불필요하게 밀려나는 역행 모션과 마지막 튕김 현상을 완전히 제거하고, 클릭 즉시 부드럽고 가볍게 슉 넘어가 사뿐히 착지하는 `cubic-bezier(0.25, 1, 0.5, 1)` 곡선을 적용해 한층 쾌적한 인터랙션을 완성했습니다.

### 🌀 2. 몰입형 스크롤 애니메이션 및 패러랙스 (Immersive Scroll & Parallax)
* 뷰포트 스크롤 도달 시 대기 상태(멀리 누워있는 3D 비스듬한 상태)에서 즉시 원위치로 솟구치며 입체적으로 안착하는 매끄러운 트랜지션 효과를 탑재했습니다.
* 우측 잡지 그리드 형태의 정보 패널과 향기 칩(Note Chips) 버튼들이 순차적으로 우아하게 솟아오르는 슬라이드 인 애니메이션을 선사합니다.

### 🖱️ 3. 커스텀 시그니처 마우스 커서 (Custom Interactive Cursor)
* 단순한 브라우저 기본 포인터를 탈피하여, 캔들의 은은한 무드를 형상화한 커스텀 마우스 포인터와 부드럽게 따라오는 동적 서클 트래커를 장착했습니다.
* 링크 및 상호작용 요소 호버 시 감각적으로 확장되는 피드백을 전달합니다.

### 📱 4. 완전 반응형 해상도 조율 (Fully Responsive Styling)
* CSS 미디어 쿼리 중첩 설정을 통해 **데스크탑(PC), 태블릿, 모바일(650px 이하)** 모든 기기에서 3D 서적 컴포넌트가 최적의 비율로 오차 없이 스케일 다운 및 센터 정렬되도록 고안되었습니다.

## 🛠️ 기술 스택 (Tech Stack)

* **Core**: React, Vite (JavaScript ES6+)
* **Styling**: Vanilla CSS (Vanilla CSS 3D Transform, Transition, Flexbox & Grid)
* **Icons**: Lucide React Icons
* **DevOps / Build**: Rollup (Vite 내장 빌드 시스템)

## 🚀 로컬 구동 방법 (Local Installation & Setup)

프로젝트를 로컬 환경에 설치하고 구동하려면 아래 명령어를 순서대로 입력하세요.

### 1. 패키지 설치
프로젝트 루트 경로에서 필수 라이브러리 및 패키지를 설치합니다.
```bash
npm install
```

### 2. 개발 서버 실행
로컬 개발 서버를 실행하여 브라우저에서 사이트를 즉시 실시간 확인합니다.
```bash
npm run dev
```
* 서버 가동 후 브라우저에서 `http://localhost:5173/`으로 접속할 수 있습니다.

### 3. 프로덕션 빌드 및 배포 사전 검증
코드를 최적화하여 압축하고 실전 배포용 파일(`dist/`)을 안전하게 빌드합니다.
```bash
npm run build
```
